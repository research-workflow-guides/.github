# Support

Use GitHub issues for project support.

## Where To Ask

- Open an issue for broken links, confusing documentation, or missing context.
- Open a pull request for small fixes that are ready to review.
- For broad changes, start with an issue before writing a large pull request.

## Helpful Details

When reporting a problem, include:

- The page or file involved
- What you expected to happen
- What actually happened
- Any browser, OS, or command output details that help reproduce the issue
